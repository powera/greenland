About WireWord Export

The WireWord export creates JSON files in the format expected by the WireWord API for language learning applications.

Export Types:
  • Split Files (recommended): Creates two files in data/trakaido_wordlists/lang_XX/generated/wireword/:
    • wireword_verbs.json - All verb conjugations
    • wireword_nouns.json - All non-verbs (nouns, adjectives, etc.)
    This is what Trakaido expects.
  • Single File: Downloads a single JSON file with all words (or filtered subset)
  • Verbs Only: Downloads just the verb conjugations file
