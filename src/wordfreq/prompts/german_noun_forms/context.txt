TASK: Generate all grammatical forms for a German noun.

FORMS TO GENERATE (2 total):

2. Plural (Plural/Mehrzahl)

GERMAN NOUN GENDERS:
- Masculine (der): e.g., der Hund, der Mann
- Feminine (die): e.g., die Katze, die Frau
- Neuter (das): e.g., das Kind, das Buch

PLURAL FORMATION:
- Add -e (der Hund → die Hunde)
- Add -er, often with umlaut (das Kind → die Kinder)
- Add -en/-n (die Frau → die Frauen)
- Add -s (das Auto → die Autos)
- Add umlaut only (der Vater → die Väter)
- No change/zero plural (der Lehrer → die Lehrer)

SPECIAL CASES:
- Some nouns have no plural (Singularetantum): e.g., der Regen, die Milch
- Some nouns have no singular (Pluraletantum): e.g., die Leute, die Eltern
- Compound nouns: Only the last component pluralizes (das Wörterbuch → die Wörterbücher)
- Foreign loanwords: May use original or German pluralization

OUTPUT FORMAT:
Provide forms with their definite articles in nominative case:
- Singular: MUST include der/die/das based on the noun's grammatical gender (CRITICAL!)
- Plural: Always use "die" for plural nominative

Examples:
- "der Hund" (singular), "die Hunde" (plural)
- "die Katze" (singular), "die Katzen" (plural)
- "das Kind" (singular), "die Kinder" (plural)

IMPORTANT: The gender (der/die/das) is an essential part of learning German nouns. ALWAYS include the correct article!

Note: German nouns are always capitalized.
